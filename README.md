# Turbo test cases

Repository Overview
---
Turbo_example collects and organizes several classic turbine/blade row cases, aiming to provide references for designers and developers with similar simulation needs. Each case typically includes: geometric design references, implementation data, representative simulation validation papers, ready-to-use mesh generation configuration files, SU2 calculation configuration files, as well as mesh and simulation results provided by the author.

Citation and Acknowledgements
---
- Each case directory contains a list of references for the geometry and validation work. When using or citing data and results from this repository, please refer to the original papers and data sources as specified in the `README.md` of each subdirectory.
- If this repository is helpful to your work, please cite the original references in your papers/reports.

Contribution Guide
---
- Contributions of new cases, configuration file corrections, or additional validation papers are welcome.
- Before submitting, please ensure the case is reproducible, necessary references are included, and the data license allows sharing.
- Please use Pull Request and describe the data source and validation status in the description.
